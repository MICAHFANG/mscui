# @msc/ui

[![license](https://img.shields.io/github/license/MICAHFANG/mscui.svg)](LICENSE)

A simple rich UI component library.

## Install

npm

```
npm install -save @msc/ui
```

yarn

```
yarn add @msc/ui
```

pnpm

```
pnpm add @msc/ui
```

## Usage

```js
import { Button } from '@msc/ui';

const App = () => <Button>Submit</Button>;
```
